# gas-inspection-system
本项目针对工业管道气体泄漏人工巡检风险高、效率低的行业痛点，基于地瓜机器人RDK X5智能计算平台，开发智能管道气体泄漏视觉巡检机器人系统。项目以RDK X5为核心控制单元，搭载视觉采集模块，依托BPU硬件加速能力运行针对管道气体泄漏场景深度优化的视觉检测模型，实现管道气体泄漏的精准识别与定位，是项目核心研发内容。机器人识别工作场景后按照预设路径完成管道巡检作业，根据识别结果触发泄漏告警，形成完整任务闭环。系统搭载交互式反馈模块，可同步输出巡检状态、泄漏告警等信息，保障巡检过程可追溯、可监控。方案充分发RDK X5算力优势，系统稳定、实用性强，适用于油气、化工等管道巡检场景。
