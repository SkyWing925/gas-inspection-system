常见问题

关于系统你可能想知道的

系统支持哪些气体类型检测？
系统通过热成像原理检测气体泄漏导致的温度异常，适用于工业压缩气体、制冷剂、蒸汽等具有温差特征的气体泄漏检测。AI 模型通过热成像灰度图识别气体扩散的形态特征，不依赖特定气体化学传感器。
检测结果 normal / warning / danger 分别代表什么？
Normal（正常）：≥80% 的采样帧未检测到运动变化，现场安全。
Warning（疑似）：存在运动变化但泄漏证据不足，建议人工复核。
Danger（泄漏）：≥30% 的帧被 DL 模型判定为泄漏，需要立即处理。
人体经过会不会触发误报？
不会。系统内置了 HumanGasDiscriminator（11 维特征判别器），通过边缘密度、紧实度、长宽比等特征区分人体与气体。人体经过会被标记为 warning（特殊情况），而非 danger（泄漏）。
如何自定义巡检路径？
使用路径 DSL 语法：f:距离（前进）、t:角度（转弯）、s（停车拍摄）。
例如："f:0.5, s, t:90, s, f:0.3, s" 表示前进 0.5m → 拍摄 → 右转 90° → 拍摄 → 前进 0.3m → 拍摄。
系统可以在哪些平台上运行？
核心系统运行在 RDK X5（ARM Cortex-A55, Ubuntu 22.04, ROS 2 Humble）。Web 控制台可在任何现代浏览器中打开，支持桌面端与移动端访问。
数据如何上传到云端？
通过 iotda_test.py 使用 MQTT over TLS 1.2 协议连接华为云 IoTDA。上传内容包括：文字告警、图片证据（base64 编码）和语音播报。
